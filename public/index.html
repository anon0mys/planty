    Introduction

    Welcome to the Planty API! You can use this API to access data about hardiness zones, keep track of your seeds, and get planting information.

    POST and PATCH Requests

    Example:

    {
  "resource": {
    "attribute": "Test"
  }
}

    Authentication

    Registration

    HTTP Request

    POST https://planty-api.herokuapp.com/api/v1/users

    Request Body

    Example Request Body:

    {
  "user": {
    "email": "test@planty.com",
    "zipcode": "80017",
    "password": "testpass123",
    "password_confirmation": "testpass123"
  }
}
    Parameter Type Description
    email string The email of the user
    zipcode string The zipcode for the user, which determines hardiness zone
    password string The password to set for the user
    password_confirmation string Must match password

    Example Response:

    {
  "id": 1,
  "email": "test@planty.com",
  "zipcode": "80017",
  "zone": "5a",
  "last_frost": "05/05/2022",
  "last_frost_short": "May 5",
  "token": "aboin130h13oginasgrtfyg35uhijok1jlrbhva"
}

    Login

    HTTP Request

    POST https://planty-api.herokuapp.com/api/v1/users/login

    Request Body

    Example Request Body:

    {
  "user": {
    "email": "test@planty.com",
    "password": "testpass123"
  }
}
    Parameter Type Description
    email string The email of the user
    password string The password for the user

    Example Response:

    {
  "id": 1,
  "email": "test@planty.com",
  "zipcode": "80017",
  "zone": "5a",
  "last_frost": "05/05/2022",
  "last_frost_short": "May 5",
  "token": "aboin130h13oginasgrtfyg35uhijok1jlrbhva"
}

    Logout

    HTTP Request

    DELETE https://planty-api.herokuapp.com/api/v1/users/logout

    Authorization Header

    {"Authorization": "Token mytokenhere"}

    Example Response:

    {
  "message": "Succesfully logged out"
}

    Seed Catalog

    Add seed to a catalog

    This endpoint adds a seed to the current user's collection of seeds.

    HTTP Request

    POST https://planty-api.herokuapp.com/api/v1/catalog

    Authorization Header

    {"Authorization": "Token mytokenhere"}

    Request Body

    Example Request Body:

    {
  "seed": {
    "id": 1
  }
}
    Parameter Type Description
    id number The id of seed to be added to the catalog

    Example Response:

    {
  "name": "Beans (Lima)",
  "tbd": "tbd",
  "planted": false
}

    Get catalog

    This endpoint returns the seeds in the current user's collection.

    HTTP Request

    GET https://planty-api.herokuapp.com/api/v1/catalog

    Authorization Header

    {"Authorization": "Token mytokenhere"}

    Example Response:

    [
  {
    "name": "Beans (Lima)",
    "tbd": "tbd",
    "planted": false
  }
]

    Errors

    The Planty API will return a errors object for any error. It uses the following error codes:

    Example Errors Response:

    {
  "errors": "This is the message for why your request failed"
}
    Error Code Meaning
    404 Not Found -- The specified resource could not be found.
    422 Unprocessable Entity -- Your request is invalid.
